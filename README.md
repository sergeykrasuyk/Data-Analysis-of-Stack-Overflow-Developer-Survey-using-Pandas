# Data-Analysis-of-Stack-Overflow-Developer-Survey-using-Pandas

## 📊 Project Overview

This project performs a comprehensive analysis of the Stack Overflow Annual Developer Survey dataset using Python and Pandas. The goal is to extract key insights into developer demographics, programming language popularity (with a focus on Python), compensation trends, remote work patterns, and learning pathways.

**Dataset:** Stack Overflow Developer Survey 2024 (Public Release)  
**Tools:** Python, Pandas, NumPy, Google Colab, Seaborn  
**Total Respondents:** 49,191

---

## 🎯 Research Questions & Key Analyses

### 1. Respondent Demographics
- **Total number of respondents:** 49,191
- **Response completeness:** 0 respondents answered all 126 survey questions (average 63.2 answers per respondent)

### 2. Work Experience Analysis
- **Mean Experience:** 13.4 years
- **Median Experience:** 10.0 years
- **Mode Experience:** 10.0 years

### 3. Remote Work Trends
- **Number of remote workers:** 10,931 (22.2% of all respondents)
- **Most common work arrangement:** Remote (10,931), followed by Hybrid (6,732)

### 4. Python Popularity
- **Overall Python usage (among those who answered):** 58.3%
- **Overall Python usage (among all respondents):** 37.5%
- **Most popular age group for Python:** 18-24 years (40.0%)
- **Least popular age group:** 65+ years (31.6%)

### 5. Compensation Analysis for Python Developers
- **Top 5 countries by average compensation:** Oman ($390,135), Andorra ($226,104), Vietnam ($218,837), USA ($173,299), Switzerland ($156,457)
- **Wide variation:** Compensation ranges from $390,135 (Oman) to $78 (Palestine)

### 6. Top Earners' Education
- **Education levels of top 5 highest-compensated respondents:**
  - Associate degree: 2 respondents
  - Master's degree: 2 respondents
  - Bachelor's degree: 1 respondent

### 7. Industry Analysis for High-Earning Remote Workers
- **Top 5 industries:** Software Development (1,186), Fintech (190), Healthcare (188), Other (176), Internet/Telecom (138)

### 8. Learning Pathways
- **Online courses users:** 10,973 respondents (22.3% of all respondents)
